# Bitasmbl-Simple-Blog-Platform-9e1728-Nodar_Mebunia

## Description
Build a web application where users can read and post blog articles. This project introduces basic CRUD operations, server-client interaction, and simple database handling for storing articles.

## Tech Stack
- React
- Node.js

## Requirements
- Display a list of blog articles for users to read
- Allow users to create new blog posts
- Enable updating and deleting existing blog posts
- Store blog articles in a simple backend

## Installation
bash
git clone https://github.com/he1snber8/Bitasmbl-Simple-Blog-Platform-9e1728-Nodar_Mebunia.git
cd Bitasmbl-Simple-Blog-Platform-9e1728-Nodar_Mebunia


## Usage
Use stack-appropriate commands to install dependencies and run React frontend and Node.js backend.

## Implementation Steps
1. Set up React client.
2. Set up Node.js server.
3. Implement article storage in backend.
4. Implement CRUD API for articles.
5. Connect React to backend.
6. Build UI for listing, creating, updating, deleting articles.

## API Endpoints
- CRUD endpoints for blog articles on the Node.js backend.
